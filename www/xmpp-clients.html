xmpp clients


the best xmpp clients

These are some of the best ones I have used. I don't use Windows or Mac or iOS, so you will have to find good recommendations for those - I won't recommend something I haven't personally used - though I will happily share positive reviews from people who have actually used the software.

The best advice - use free and open source software! xmpp is a great example of software that is owned by no-one and decentralized, able to be run by anyone, anywhere, for any reason.



Look for xmpp clients that support OMEMO encryption - this will help you make sure your conversations with your chat partners are encrypted and private on both ends.



~ ~ ~ ~ ~

Gajim - Linux

This is a great xmpp client for Linux distributions that comes packaged in most distro's standard repositories.

It is lightweight, classic-looking, and extensible with PGP and OMEMO encryption.

To get it, just run:

sudo apt install gajim

in Debian/Ubuntu based distros, or:

sudo pacman -Sy gajim

For Arch based distros.


~ ~ ~ ~ ~

Pidgin - Linux, BSD, Windows

Pidgin is an older and well-known cross-platform application that supports multiple types of chat protocols - XMPP included. With support for enabling OMEMO encryption with the lurch plugin, you can use Pidgin with xmpp privately.

You can download Pidgin from their site at the link above, or you can grab it from most distribution repositories by simply running:

sudo apt install pidgin

For Ubuntu and Debian based distros, or:

sudo pacman -Sy pidgin

On Arch-based distros.


~ ~ ~ ~ ~

Conversations - Android

Conversations is one of the premier xmpp apps for Android. Even better, it is available from the F-Droid appstore, a store full of nothing but free and open-source applications.

monocles chat - Android

Monocles Chat is an xmpp client forked from Conversations, and is available from the F-Droid app store. You can use it to sign into any xmpp server (like civitaz!), the UI is slick, and it supports all of the same encryption schemes as Conversations (OMEMO and PGP).


~ ~ ~ ~ ~

Profanity - Linux, BSD, macOS, Android

Profanity is a console-based xmpp client. It is a very slick client if you want to go the old-fashioned way and chat right in your terminal.

It is available cross-platform, able to be used on Linux, BSD, macOS, and even Android through a Termux package.

Learn more about Profanity by checking out the docs to see how you can use it on your own system for terminal-based communications.


If you use another system and can't find a recommendation for your platform listed here, no worries - there are some helpful resources right over at the official xmpp site where you can learn more about other xmpp clients for all platforms.
